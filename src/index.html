Cookie Generator for acw_sc__v2

Quick Start

Input html file, output acw_sc__v2 cookie. Check GitHub for source code and more details.

Example

curl https://www.example.com/ --cookie 'acw_sc__v2=65afac17e880921014c4ead657413970d8b23ccb'

Python SDK

To use the Python SDK, you need to install it via pip: 

pip install acw-sc-v2-python-sdk
Then you can use it in your Python code like this: 
import acw_sc_v2
cookie = acw_sc_v2.generate_cookie(html_content)

GoLang SDK

To use the GoLang SDK, you need to get it with go get: 

go get github.com/WangYihang/acw-sc-v2-go-sdk
Then you can use it in your Go code like this: 
package main

import (
    "fmt"
    "github.com/WangYihang/acw-sc-v2-go-sdk"
)

func main() {
    cookie, err := acwscv2.GenerateCookie(htmlContent)
    if err != nil {
        fmt.Println("Error:", err)
        return
    }
    fmt.Println("Cookie:", cookie)
}
